# 📰 AI 自动新闻摘要 Agent

基于 **LangGraph** + **大模型** 实现的全自动新闻抓取、分类、摘要及日报生成工具。

---

## 📖 项目背景

每天新闻信息繁杂，人工阅读效率低。本项目通过 AI 自动化完成以下流程：

> **新闻抓取 → 去重 → 清洗 → AI分类 → AI摘要 → 生成日报 → 邮件推送**

实现无人值守的每日新闻简报，让信息获取更高效。


## ️ 项目架构
graph TD
    A[RSS新闻源] --> B[异步网络抓取]
    B --> C[文章去重]
    C --> D[文本清洗]
    D --> E[AI分类<br/>(带缓存)]
    E --> F[AI单篇摘要<br/>(带缓存)]
    F --> G[AI分类汇总<br/>(带缓存)]
    G --> H[Markdown日报生成]
    H --> I[自动邮件发送]

##  项目功能
- ✅ **自动抓取**：从 RSS 源自动获取每天最新新闻
- 🧹 **数据清洗**：自动去重、清洗无效文本
- 🏷️ **智能分类**：AI 自动分类（集成缓存机制，避免重复消费）
- 📝 **自动摘要**：AI 生成单篇及分类汇总摘要（带缓存）
- 📊 **日报生成**：自动生成 Markdown 格式的日报
- 📧 **邮件推送**：支持定时自动发送日报到指定邮箱
- ⏱️ **定时运行**：支持后台定时任务

## 项目亮点
- **LangGraph 流程编排**：基于有向图结构实现全流程精准控制，支持状态管理与错误重试，确保新闻抓取、去重、清洗、AI分类、AI摘要、报告生成、邮件推送等环节有序执行。
- **智能缓存优化**：对 AI 分类、AI 摘要环节加入本地缓存（cache.json），相同标题/内容不重复调用大模型，大幅降低 Token 消耗。
- **全自动无人值守**：从 RSS 新闻源抓取、处理到最终日报邮件推送，全流程无需人工干预，支持每日定时自动执行。
- **结构化日报输出**：自动生成分类清晰、格式美观的 Markdown 日报，按主题汇总新闻，便于快速阅读与归档。
